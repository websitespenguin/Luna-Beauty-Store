# Luna Beauty Store - Demo

Página web demo de una tienda de cosméticos hecha con:

- HTML
- CSS
- JavaScript

## Cómo usarla

1. Abre `index.html` en tu navegador.
2. Para cambiar el número de WhatsApp, abre `script.js` y modifica esta línea:

```js
const phoneNumber = "573001234567";
```

3. Reemplaza las imágenes en la carpeta `assets` si quieres usar fotos reales.
   Mantén los nombres:
   - producto-1.svg
   - producto-2.svg
   - producto-3.svg
   - producto-4.svg
   - producto-5.svg
   - producto-6.svg
   - producto-7.svg
   - producto-8.png

## Nota

El proyecto incluye imágenes de demo para que la página funcione completa.
